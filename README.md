# Secure Weather API

This is a REST API project built using Node.js and Express.

## Features
- Authentication using JWT
- Secure protected routes
- Weather API integration
- Error handling

## Tech Stack
- Node.js
- Express.js
- JWT (JSON Web Token)

## Run Project Locally

1. Clone the repository
git clone https://github.com/rajgupta0807/xml-project

2. Go to project folder
cd secure-weather-api

3. Install dependencies
npm install

4. Start server
node server.js

## API Endpoints

### Register User
POST /register

### Login User
POST /login

### Get Weather (Protected Route)
GET /weather

## Project Structure
- server.js → main file
- package.json → dependencies
- routes → API routes
